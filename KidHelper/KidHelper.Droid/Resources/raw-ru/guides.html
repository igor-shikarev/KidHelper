Руководство

После запуска приложения, необходимо настроить список страниц. Перейти в список можно через меню «Настройка страниц».

Нажав на кнопку «Добавить», добавить новую страницу.

В открывшемся диалоге, можно выбрать отдельно для левой и правой сторон: фотографию предмета, записать голос для фотографии (кнопка «Запись») и выбрать цвет рамки для фотографии.

Поле сохранения должно получиться что-то подобное.

При одиночном нажатии на строку, вызываестя диалог редактирования страницы. При долгом нажатии на строку, вызывается диалог удаления страницы.

В результате страница будет выглядеть, как на рисунке ниже. При одиночном нажатии на фотографию, будет проигрываться записанный голос.
